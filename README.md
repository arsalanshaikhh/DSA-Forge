# DSA Forge

> **Learn Patterns · Practice Problems**

A modern, interactive web-based platform for mastering Data Structures and Algorithms (DSA) through structured learning and hands-on problem solving.

## 🎯 Overview

DSA Forge is designed to help developers and computer science students strengthen their foundational knowledge of Data Structures and Algorithms. With an intuitive interface, curated problem sets, and progress tracking, it transforms the way you prepare for technical interviews and coding challenges.

## ✨ Features

- **Dashboard View** — Comprehensive overview of your learning progress with statistics and category-based organization
- **Interactive Problem Lists** — Browse and filter problems by difficulty level (Easy, Medium, Hard)
- **Pattern-Based Learning** — Organized by common DSA patterns (Arrays, Strings, Trees, etc.)
- **Progress Tracking** — Visual progress indicators showing your completion status for each pattern
- **Code Editor Integration** — Built-in CodeMirror editor with syntax highlighting for writing and testing solutions
- **Dark Theme** — Eye-friendly, modern dark interface optimized for extended coding sessions
- **Difficulty Filtering** — Quickly find problems matching your skill level
- **Pattern Categories** — Well-organized curriculum covering essential DSA concepts

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning the repository)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/arsalanshaikhh/DSA-Forge.git
cd DSA-Forge
```

2. Switch to the development branch (if needed):
```bash
git checkout v4
```

3. Open the application:
   - Simply open `index.html` in your web browser, or
   - Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```
   - Then navigate to `http://localhost:8000`

## 📚 How to Use

### Navigation

1. **Dashboard** — Start here to see your overall progress and available patterns
2. **Problem Categories** — Click on any pattern to view related problems
3. **Select a Problem** — Choose a difficulty level and problem to practice
4. **Code Editor** — Write your solution in the integrated editor
5. **Submit & Track** — Submit your solution and track your progress

### Learning Path

- Start with **Easy** problems to understand fundamental concepts
- Progress to **Medium** problems to consolidate your knowledge
- Challenge yourself with **Hard** problems for mastery
- Revisit patterns and problems to reinforce learning
